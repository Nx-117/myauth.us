myauth.us
=========

Battle.net Authenticator Online/战网安全令在线版

如欲安装在自己的网站上请按照如下步骤安装

- - -
①修改/includes/config.php，各项注释都已写明<br>
②建立auth数据库，导入auth.sql数据库结构<br>
③修改代码中一些与联系邮箱等，导入自己的key，你改吧，随便改，记得遵照协议就行了，自己建好站了请别把我的名字都删了好不，留个XX别人就看不到了么
- - -
可以愉快的开放了<br>
